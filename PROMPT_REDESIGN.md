# Prompt de Rediseño Premium - Frontend Senior UI/UX

Actúa como un Ingeniero de Frontend Senior experto en UI/UX. Quiero que rediseñes por completo mi página web para que tenga un aspecto visual premium, moderno y profesional, y que funcione al 100% sin errores de carga.

Para lograrlo, aplica estrictamente las siguientes reglas:

1. **ESTILOS INYECTADOS**: Utiliza Tailwind CSS cargado a través de su CDN oficial en la cabecera (`<script src="https://cdn.tailwindcss.com"></script>`) para asegurarte de que todo el diseño visual se aplique de inmediato sin depender de archivos CSS externos míos.

2. **ESTILO VISUAL PREMIUM**: Aplica una paleta de colores sofisticada (por ejemplo, un modo oscuro elegante con tonos gris pizarra/zinc y detalles en un color de acento llamativo como azul eléctrico o violeta, o un modo claro limpio y minimalista). Usa esquinas redondeadas (`rounded-xl`), sombras suaves (`shadow-lg`) y transiciones fluidas en los botones (`transition-all duration-300 hover:scale-105`).

3. **ESTRUCTURA COMPLETA**: No me des fragmentos de código ni comentarios como "// aquí va el resto". Devuélveme el código HTML y CSS completo y estructurado, listo para copiar y reemplazar en mi archivo actual.

4. **RESPONSIVO**: Asegúrate de que el diseño sea totalmente responsivo usando las clases `md:` y `lg:` de Tailwind para que se vea perfecto tanto en celulares como en pantallas de escritorio.

5. **SIN ERRORES**: No utilices librerías externas de JavaScript ni frameworks complejos que puedan romperse. Toda la interactividad visual (como efectos al pasar el mouse) debe resolverse puramente con clases de Tailwind.
